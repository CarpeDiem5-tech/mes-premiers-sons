# Mes premiers sons

## Règle UX — narration pour enfants non lecteurs

Toute narration destinée à un enfant non lecteur doit être jouée à l'oral.

Elle ne doit pas être affichée à l'écran, sauf si le texte constitue lui-même l'objet de l'apprentissage : lettres, syllabes, mots ou phrases.

L'écran doit toujours privilégier les éléments visuels et interactifs plutôt que de longs textes.
